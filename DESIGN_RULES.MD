# DESIGN RULES

Use:
- asymmetrical layouts
- dark sections mixed with light photography
- cinematic spacing
- layered typography
- large image transitions
- subtle borders
- premium hover states

Avoid:
- boxed layouts
- overused cards
- full-width meaningless sections
- generic icons
- tiny text
- clutter

Preferred layout rhythm:
1. Large hero
2. Visual statement
3. Interactive services
4. Location credibility
5. Supplier trust
6. Projects
7. CTA

All sections must feel intentional.
