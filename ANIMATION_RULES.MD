# MOTION SYSTEM

Motion should feel:
- smooth
- architectural
- confident
- expensive

Use:
- fade-up transitions
- stagger reveals
- image parallax
- smooth hover transforms
- text masking reveals
- scroll-triggered transitions

Avoid:
- bouncing
- elastic animations
- flashy motion
- fast transitions

Target:
60fps smooth scrolling.

Preferred libraries:
- Framer Motion
- GSAP
- Lenis
